								--Linux基础--
1. 操作系统的作用：将一系列对计算机硬件的操作封装成系统调用供系统安装的软件调用，简化用户操作

2. Linux内核：系统调用和终端命令，系统调用调用系统硬件，终端命令调用系统调用间接调用系统硬件

3. 终端命令的格式：commond [-option] [parameter]

4. 查看命令帮助：command --help 或者 man command

5. 文件和目录常用命令
	1>.查看目录内容：ls
		Linux 系统下面的隐藏文件以.开头，使用ls时不显示隐藏文件，单独的点表示当前目录，..表示上级目录
		ls -a 可以显示所有的文件，包括隐藏文件
		ls -l 以列表的形式显示文件的详细信息
		ls -l -h 以人性化的方式显示文件的大小，-h需要配合-l使用，单独使用没有效果
		ls a* 可以使用通配符列出需要查找的文件，*代表任意多个任意字符，？代表一个字符，必须是一个字符
	2>.切换目录：cd
		cd 和 cd ~表示回到用户的家目录，cd . 保持当前目录不变，cd ..返回上级目录。cd - 可以在最近两次工作目录之间来回切换
	3>.创建和删除：touch, mkdir, rm
		touch 会创建一个空白文件， 如果文件已经存在，则修改文件的修改时间
		mkdir 创建一个目录， mkdir -p 会递归创建目录，Linux下同一个目录下文件和目录是不能重名的
	4>.拷贝和移动，删除文件：cp, mv, rm
		rm 命令删除文件，该命令会直接将文件删除，删除后文件不能恢复
		rm -r 用来删除目录
		rm -f 强制删除文件，如果文件存在直接删除，如果文件不存在也不会给任何提示
		rm *.txt rm命令也可以使用通配符，用法与ls中的一样
		cp source taget 复制文件 -r选项可以复制文件夹，如果目标文件已经存在不给提示，-i选项可以在目标文件存在时提示是否覆盖
		mv source target 移动文件或目录，也可以重命名文件或目录，使用mv重命名文件时如果目标文件存在则会覆盖目标文件，可
		以用-i选项可以在目标文件存在时提示是否覆盖
	5>.查看文件内容：cat, more, grep
		cat和more都可用来查看文件内容，cat全部显示，可以用来创建文件，文件合并，追加文件内容。
		more分页显示, f向前滚动一屏，空格向前滚一页，b向后回滚一屏
		cat 有两个选项，cat -b 对非空输出行编号， cat -n 对所有行编号
		grep 强大的文本搜索工具，可以用来在文件中搜索文本
		grep [option] pattern [file] 可用的选项有-n显示行号, -i忽略大小写，-A 10表示打印之后10行，-B 10表示打印前面10行,
		-C 10表示打印前后10行, -v表示取反, 即不包含pattern的行，-w强制完全匹配。当需要匹配的pattern中有特殊字符时需要用""或者''
		grep允许对文件进行模式查找，所谓模式就是正则表达式比如^$
	6>.其他：tree,echo, >, >>, |
		tree [dir] 以树形结构列出当前文件目录结构
		tree -d 只显示目录
		echo 结合重定向可以给文件中写或者追加内容
		|管道，Linux系统中可以通过|将前面命令的输出作为后面命令的输入

6. 远程管理常用命令
	1>. shutdown [option] [time] 关机或者重启系统，-r参数表示重启，不指定时间表示1分钟之后关闭或者重启电脑
		shutdown -r now 其中now表示现在重启电脑
		shutdown -c 可以取消之前指定的关机计划
		shutdown +10表示10分钟之后重启系统
	2>. ifconfig 查看或者配置计算机中的网卡配置信息
		ping ip 检测到ip地址的连接是否正常。ip地址是设置在网卡上的地址信息，不能重复
		
		查看计算机上面的ip地址：
		ifconfig |grep -w 'inet'|grep -v '127.0.0.1'|awk '{print $2}'  查看本地的ip地址
		ping 127.0.0.1 本地回环地址，检查本地网络是否畅通
	3>.SSH客户端是一种使用Secure Shell协议链接到远程计算机的软件程序，可以有效防止远程登录过程中的信息泄露
		SSH服务器的默认端口22
		Web服务器默认的端口80
		HTTPS服务器默认的端口443
		FTP服务器默认的端口21
		ssh [-p port] user@remote 其中user是远程机器上的用户名，remote是远程机器的地址，port是SSH Server监听的端口，默认22可以省略
		ssh终端命令只能在Linux系统或者Unix系统下面使用，Windows系统下面需要安装ssh远程客户端，比如Putty或者XShell都是常用的SSH客户端
		第一次连接SSH服务器时会提示是否授权，点击授权后会在客户端家目录下面的.ssh文件夹下面生成一个known_hosts文件，将服务器信息保存
		到当前这个文件中，下次再使用ssh远程连接的时候就不提示授权
		
		ssh免密设置：
		1)执行ssh-keygen可以生成一对秘钥，一路回车就好
		2)执行ssh-copy-id user@remote 将生成的秘钥添加到远程机器
		原理：生成的秘钥使用的是非对称加密算法，使用公钥加密的数据需要使用私钥进行解密，使用私钥加密的数据需要使用公钥进行解密。
		客户端将数据使用私钥进行加密，然后发送到服务器端，服务器端接收到数据之后使用公钥进行解密查看发送的内容，服务器端将需要
		返回的结果再使用公钥进行加密返回给客户端，客户端使用私钥进行解密查看数据，所以不需要使用密码进行认证
		
		ssh配置别名：
		在.ssh/config文件中添加以下内容：(若文件不存在先创建config文件)
		Host name
			HostNmae ip地址 
			User username
			Port 22
	4>. scp secure cp远程拷贝文件
		scp [option] source target  -P可以指定端口，-r选项可以复制文件夹
		scp 11.py user@remote:Desktop/11.py  将本地文件拷贝到远程服务器，不以/开头表示相对路径，这里的相对路径是相对用户的家目录
		scp user@remote:Desktop/22.py 22.py  将远程服务器上面的文件拷贝到本地文件系统的当前目录
	5>. FileZilla是Windows系统下面使用的一种FTP客户端，可以将文件上传到远程服务器，使用的是FTP协议而不是SSH，默认端口号21
	
7. 系统权限相关命令
	1>. ls -l 可以详细显示文件或文件夹的权限。
		权限：drwxrwxr-x 第一位表示文件类型，d表示文件夹，后面九位分别表示用户的权限用户组的权限
			其他组的权限。
		硬链接数：第二列表示文件或文件夹的硬链接数，即有几种方式可以到达当前文件夹或文件，文件夹至少有2个硬链接数，分别是绝对
			路径和.，因为cd .可以在当前路径下进入到当前路径
		拥有着：当前文件或文件夹的拥有着
		组：当前文件或文件夹所在的组
		大小：文件的大小
		时间：文件创建或者最后修改的时间
		名称：文件或文件夹的名称
	2>. chmod  可以修改用户/组对文件/文件夹的权限
		chmod +/-rwx 文件/文件夹  可以给文件或者文件夹增加或者减少权限
		chmod [a,g,u,o]+/-rwx  文件夹/文件  a表示修改用户，组和其他组的权限，g表示给组修改权限，u表示给用户修改权限
		文件夹的可执行权限：
		对文件夹来说进入文件夹就相当于执行了该文件夹，在Linux系统中要对一个文件夹执行终端命令，比如cd, ls 那么这个文件夹必须要有
		可执行的权限，当文件夹有了可执行权限时可以cd到这个文件夹下面，如果该文件夹没有可读权限，那么在该文件夹里面执行ls命令将会
		出错，报权限不足。文件夹的可写权限表示可以修改文件夹中的内容，比如创建文件
	3>. root用户是系统的管理用户，对系统的所有资源具有所有权限
		sudo用来以其他身份执行命令，预设的用户是root，用户使用sudo时需要输入密码，之后有5分钟的有限期
	4>. 组管理命令：
		1) groupadd 组名  增加组名
		2) groupdel 组名  删除组名
		3) chgrp 组名 文件/文件夹  修改文件或文件夹所属的组
		查看组信息：cat /etc/group 
		标准用户没有添加和删除组的权限，在执行时可以在命令前夹sudo命令用root身份执行命令
	5.> 用户管理命令：
		useradd -m -g 组 用户名  新建一个用户，-m会自动创建该用户的家目录，-g会指定该用户所属的组，不指定组时会自动创建于与用户
		名重名的组名称。不加-m参数时不会自动创建用户的家目录，需要重新创建用户家目录
		passwd 用户名  用来给用户设置密码，没有密码的用户是没办法登录系统的
		确认用户信息：cat /etc/passwd grep|username 即用户信息保存在/etc/passed文件
		/etc/passwd 文件中的内容：第一列是用户名称，第二列是用户密码，x表示加密，第三列表示用户ID，第四列表示组ID，第五列表示全名，
		第六列表示用户家目录，第七列表示登录之后使用的终端命令
		userdel -r 用户名  删除用户, -r选项会自动删除用户的家目录
	6.> id [用户名]  可以查看用户信息，不指定用户名则查看当前登录用户的用户信息
	7.> who  查看当前所有登录的用户列表，可以查看多用户登录系统中已经登录的所有用户列表
	8.> whoami  获取当前登录用户的账号名
	9.> usermod  用来修改用户的主组或者附加组以及登录shell
		主组：一般在新建用户时指定，在/etc/passed文件第四列GID对应的组
		附加组：在/etc/group 文件中的最后一列，表示该组的用户列表，用于指定用户的附加权限
		usermod -g 组 用户名  修改用户的主组
		usermod -G 组 用户名  修改用户的附加组
		usermod -G 组 -a 用户名  将用户添加到新的附加组，不会删除原有的附加组
		usermod -s /bin/bash  修改用户登录的shell，修改用户登录的shell为/bin/bash时可以正常使用,若为/bin/sh则无法使用tab和上下键，文件
		和文件夹也不会高亮显示
	10.> which 该命令用于查找命令执行的位置
	11.> su [-] username  切换用户，其中-可以切换到用户家目录，否则不加-则切换用户后保持原目录不变
	12.> chown 修改文件的拥有者
	
8. 系统信息相关命令
	1.> date 查看当前系统时间
	2.> cal 查看日历，-y选项可以查看一年的日历
	3.> df -h  显示磁盘剩余空间，-h以人性化的方式显示文件大小
	4.> du -h [目录]  显示目录下的文件大小
	5.> ps [aux]  ps默认只会显示当前用户通过终端启动的进程，ps aux 可以查看正在运行的所有进程状态   
	6.> top 动态显示当前系统中CPU占有率和内存占有率较高的进程
	
9. 其他命令
	1.> find 命令功能非常强大，通常用来在特定目录下搜索符合条件的文件
		find [路径] -name "*.py" 不指定目录时在当前目录下面查找
	2.> ln -s 被链接的源文件 链接文件  -s选项建立软链接，否则建立的是硬链接，建立软连接时链接的源文件使用绝对路径，保证链接文件移动后能
	够正常使用。当建立硬链接时，如果删掉源文件，硬链接的文件内容不会发生变化，而软连接删除源文件后则不能链接到源文件
	在Linux系统文件名和文件数据是分开存储的
	3.> tar -cvf 打包文件.tar 目标文件/目录
	4.> tar -xvf 解包文件.tar -C 解压目录  解压文件到指定目录，其中-z可以调用gzip，以压缩tar包，文件以.tar.gz结尾。tar只会打包文件，不
	会压缩文件，tar -czvf可以先打包文件，然后调用gzip压缩文件。解压缩时使用tar -xzvf.这些命令中的-f选项必须放在最后面
	-C选项指定的文件夹必须存在
	5.> tar 和bzip2结合也可以实现打包和压缩，可以使用-j选项取代-z选项，文件以.tar.bz2结尾
	6.> lsof -i:端口    查看端口是否被占用
10. 软件安装
	apt 是Advance Package Tools，是Linux系统下一款安装包管理工具，可以在终端安装，卸载和更新软件包
	安装: sudo apt install 软件包
	卸载：sudo apt remove 软件名
	更新：sudo apt upgrade   
	命令中的sudo可以让系统中的所有用户都可以使用安装的软件

	2.> tar -tf test.tar  列出tar文件的内容，-v参数可以显示文件的详细信息
	3.> zip -sf test.zip  列出zip文件的内容

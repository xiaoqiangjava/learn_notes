						HBase shell学习笔记
1. 启动: bin/hbase shell 

2. 创建表:
	create '表名称', '列族名称1','列族名称2','列族名称N'
	默认创建的表只能存储一个version的数据, 需要存储一个数据的多个版本的话, 创建表的时候需要指定version
	create '表名称', {NAME => '列族名称1', VERSIONS => 3}

3. 查看表:
	list 获取所有表名; describe '表名称' 查看表的详细信息

4. 添加记录:
	put '表名称', '行键', '列族名称:列名称','列的值'[,'时间戳']
	
5. 查看记录: 
	get '表名称', '行键' 
	get '表名称', '行键', {NAME => '列族名称1', VERSIONS => 2}  # 当一条记录有多个版本时可以指定版本数

6. 查看记录总数:
	count '表名称' [, CACHE=>1000]

7. 扫描记录:
	scan '表名称'

8. 扫描记录:
	scan '表名称' , {COLUMNS=>'列族名称:列名称'}   # 当列只有一个时也可以使用COLUMN属性
	scan '表名称', {COLUMNS => '列族名称', VERSIONS => 3}    # scan操作时指定记录的版本数
	只指定列族, 不指定列时, 会扫描该列族下面的所有列

9. 更新记录:
	就是使用put重写一遍进行覆盖, 有多个版本时不会覆盖前面的值

10. 删除记录
	delete '表名' ,'行键' , '列名称'

11. 删除记录: 删除一行记录
	deleteall ‘表名’,‘行键’

12. 删除一张表:
	第一步 disable '表名称' 第二步 drop '表名称
	disable_all 'regex'   禁用满足正则表达式的所有表
	drop_all 'regex'      删除满足正则表达式的所有表

13. 截断表:
	truncate '表名'  清空表中记录, 删除region, 新建region
